# Weyl Boundary

This repository contains the symbolic computational framework accompanying the study of moving boundary problems for parabolic partial differential equations.

The goal of the project is to develop a Weyl-algebra approach to characterize moving boundaries through compatibility matrices derived from transported differential operators.

## Current status

- ✅ Weyl algebra implemented.
- ⏳ Heat transport module under development.
- ⏳ Boundary compatibility module.
- ⏳ Matrix construction.
- ⏳ Symbolic determinant computation.
